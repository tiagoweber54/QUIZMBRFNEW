🔄 Sincronização Manual
A sincronização acontece automaticamente a cada mudança. Use os botões acima somente se necessário.
📖 Como configurar (5 minutos)
  1. Acesse firebase.google.com → Criar projeto → Nome: MBRF-Quiz
  2. Menu esquerdo → Realtime Database → Criar banco → Modo de teste → Habilitar
  3. ⚙️ Configurações → Seus apps → ícone </> → Registrar app
  4. Copie o databaseURL e o apiKey do bloco firebaseConfig
  5. Cole os campos acima e clique Salvar e Conectar
  6. Cole as regras no Firebase: { "rules": { ".read": true, ".write": true } }
